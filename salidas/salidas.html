SALIDAS EN JS

JavaScript tiene varias formas de mostrar información en pantalla. Aquí algunas de las más comunes:

  • console.log(): Imprime mensajes en la consola del navegador. 
    console.log("Hola, mundo!");
  • alert(): Muestra un cuadro de alerta en la ventana del navegador. 
    alert("Hola, mundo!");
  • document.write(): Inserta texto directamente en el documento HTML. 
    document.write("Hola, mundo!");
  • innerHTML: Modifica el HTML de un elemento específico. 
    document.getElementById("miElemento").innerHTML = "Hola, mundo!"
  • console.error() y console.warn(): Para mostrar mensajes de error o advertencias en la consola. 
    console.error("Esto es un error");
    console.warn("Esto es una advertencia");
