Variables en JavaScript

  • Las variables ej JS pueden ser de tipo 'var', 'let, o 'const', por ejemplo:

                        var nombre = "Luis";
                    let apellido = "Santos";
                    const pi = 3.14;

  • 'var' la usamos para declarar variables globales

  • 'let' para declarar variables locales
