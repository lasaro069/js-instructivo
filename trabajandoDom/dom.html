Selectores




Sirven para seleccionar elementos del html y almacenarlos en variables

Estos selectores son 'métodos' por tal motivo deben ser aplicados a elementos

Exsiten dos clases: individuales y múltiples




Selectores Individuales


  • getElementById(): permite seleccionar un elemento por su 'id'

    Creamos la variable boton y en el () colocamos el 'id' al que queremos acceder

                    var boton = document.getElementById('boton');
  • querySelector(): Selecciona el elemento con un selector CSS, devuelve el primer elemento que coincida 
                var segundoParrafo = document.querySelector('#segundo');

    Este selector se utiliza con '#' como se hace en css




Selectores Múltiples


  • getElementsByClassName: Selecciona todos los elementos que tengan la clase especificada 
                    var parrafos = document.getElementsByClassName('texto');
  • getElementByTagName(): Selecciona todos los elementos con la etiqueta seleccionada 
                var etiquetas = document.getElementsByTagName('p');

    Este selector se utiliza con las etiquetas del html

  • querySelectorAll(): contrario al otro 'queryselector', este selecciona todos los elementos que coincidan 
                    var todosParrafos = document.querySelectorAll('.texto')



Crear Elementos en el Html


  • Para crear elementos o 'nodos' utilizamos el método document.createElement('elemento que se quiere crear') 
                    var parrafoNuevo = document.createElement('p'); /* Creamos un nuevo párrafo */
  • para crear elementos de texto utilizamos el método 'document.createTextNode('texto creado');' 
                    var textoNuevo = document.createTextNode('Soy un nuevo texto');  /* Creamos un nuevo texto */
  • para agregar el nuevo texto al nuevo párrafo utilizamos el método '.appendChild()' 
                    parrafoNuevo.appendChild(textoNuevo);  /* insertamos el texto nuevo en el párrafo nuevo */
  • Para agregar un nuevo atributo al elemento creado utilizamos el método '.setAttribute'. Recibe dos parámetros: el atributo y el valor 
                    parrafoNuevo.setAttribute('class', 'texto');    /* agregamos el atributo 'class' con el valor 'texto' */
  • para seleccionar el sitio donde va a ser agregado el nuevo elemento utilizamos 'document.getElementsByClassName('nombre del elemento contenedor')[posición del elemento contenedor];' 
                    var contendedor = document.getElementsByClassName('contenedor')[0]
  • Para agregar el elemento dentro del contenedor seleccionado utilizamos el método '.appendChild()' 
                    contendedor.appendChild(parrafoNuevo)



Ubicar elementos en el html creados en JS


Si el contenedor padre no tieme un identificador preocedemos de las siguiente forma ya que no podríamos aplicar el '.appendChild()':

  • Creamos una variable y seleccionamos un identificaodr del elemento hijo del contenedor padre y su posición 
                    var primerParrafo = document.getElementByClassName('texto')[0]
  • Accedemos al elemento padre de ese párrafo 
                    var parrafoPadre = primerParrafo.parentNode;
  • Si no tiene un atributo, podemos asignarselo 
                    parrafoPadre.className = 'contenedor'   // asignamos el atributo 'clase' con el valor 'contenedor'
  • Para ubicarlo en una posición específica dentro de este contenedor padre, por ejemplo, en la primera posición:

    seleccionamos el elemento padre y le colocamos el método .insertBefore para insertar el nuevo elemento al contenedor padre

    Este método recibe dos parámetros:

    la variable creada con el nuevo elemento

    la variable creada con el elemento siguiente donde quiero ubicar el nuevo elemento

                    parrafoPadre.insertBefore(parrafoNuevo, primerParrafo);

Agregar o modificar clases o ids a elementos del html


creamos una variable con el elemento que vamos a modificar

        var nombreVariable = document.getElementById('nombreSelector')

Para modificar la clase accedemos a la variable con la propiedad '.className

        nombreVaraible.className = 'nuevoSelector;'

Para agregar un 'id' a un elemento

        var primerParrafo = document.getElementsByClassName('texto')[0];

        primerParrafo.id = 'primero';
