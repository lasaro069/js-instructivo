Números Aleatorios

Este objeto genera un número aleatorio entre 0 y 1 utilizando el método random

nomenclatura:

        var numeroAleatorio = Math.random();

        console.log(numeroAleatorio)

Si queremos un número mayor que 1, multiplicamos el resultado de la variable por el número o rango deseado

        var numeroAleatorio = (Math.random()*10);

        console.log(numeroAleatorio);

Podemos agregar unos métodos a Math

  • Math.floor(): Redondea el número decimal al número entero inmediatamente menor 
                    var numeroAleatorio = Math.floor(Math.random()*10);
  • Math.ceil: Redondea al número entero mayor más cercano 
                    var numeroAleatorio = Math.ceil(Math.random()*10);
  • Math.round: redondea al número más cercano
