Funciones, Arreglos, Objetos

Funciones

Son piezas de código que pueden ser reutilizadas una y otra vez

Las funciones pueden ser con parámetros y sin parámetros

Funciones sin Parámetros

Lo primero que hay que hacer es Declarar la función

        function saludar(){
            alert('Hola Mundo');
        };

Dentro de las llaves se colocan las tareas que la función va a desempeñar

Para ejecutarla escribimos:

        saludar();

Funciones con parámetros

La función con parámetros lleva, dentro del paréntesis los parámetros deseados

        function suma(num1, num2){
            var resultado = num1 + num2;
            console.log(resultado);
        }

Para ejecutarla escribimos:

        suma(23, 5);

Funciones con Retorno

return llama a la variable del cuerpo de la función y le da un valor

        function sumar(num1, num2){
            var resultado = num1 + num2;
            return resultado;
        }

Objetos

Un Objeto es una función que tiene propiedades y métodos

Para declarar un objeto:

        var planta = {
            color: 'verde';
            tamano: 'mediano';
            cantidad: 2;
        }

Para acceder a las propiedades utilizamos

        object.propiedad;

Para acceder a los métodos:

        object.metodo();

Ejemplo:

        var texto = 'Los ensayos cortos son escritos en los que se analiza y comenta un concepto, una idea o un asunto significativo de manera relativamente breve. En estos textos, el autor suele hacer explícita su perspectiva a través del planteo de una hipótesis argumentada. Por ejemplo: una tesis, una monografía.';

Propiedades y métodos

Las propiedades se definen como:

var nombreVariable = elememto.propiedad;

Los métodos se definen como:

var nombreVariable = elemento.metodo();

  • length: Muestra el número de caracteres de una variable 
                    var numeroCaracteres = texto.length;
  • toUpperCase(): Transforma tods los caracteres a mayúsculas 
                var minus = texto.toUpperCase();
