Para insertar comentarios de bloque utilizamos /* al inicio del bloque y */ al final del bloque

  • Ventana Emergente 
                    alert("Hola Mundo");
  • document.write

  • console.log: Salida desde la consola de Js 
                    console.log("Hola Mundo")

Variables

Pueden ser declaradas como var o let

        var nombre = "Luis";
        let nombre2 = "Felipe";

Los tipos de variables en Js son:

  • Numéricas: 
                    var numeroEntero = 23;
                var numeroDecimal = 23.6;
  • Cadenas de Texto 
                    var texto = 'soy una cadena de texto'
  • Booleana 
                    var bool1 = true;
                var bool2 = false;
  • Elementos Html 
                    var parrafo = document.querySelector(nombre del 'id' del elemento en html);
  • Arreglos 
                    var familia = ['Luis', 'Leidi', 'Duvan', 'Felipe'];
  • Objetos 
                    var planta = {
                    color: 'verde', 
                    tamano: 'grande', 
                    cantidad: 3
                };
  • Funciones 
                    var miFuncion = function(){};

Métodos con Strings

        const producto = "Monitor de 20' ";

         Para visualizar el contenido de una variable en la consola:
        console.log(producto);

        para visualizar la longitud de la variable:
        console.log(producto.length)

        para verificar si una cadena de texto se encuentra dentro de una variable:
        console.log(producto.indexOf('monitor'))
        devuelve la posición en la que se encuentra el elemento

        a diferencia del método includes que devuelve un booleano:
        console.log(producto.includes('monitor'))
