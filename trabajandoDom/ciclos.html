Ciclos

Ciclo 'for'

Permite repetir un código mientras una condición sea true.

        for(variable inicio; condición; incremento){
            código que se repite;
        }
    
        imprimir los números del 1 al 10

        for(var i = 1; i <= 10; i++){
            console.log(i);
        }

Ciclo 'for' para recorrer arreglos

        var semana = ['lunes', 'martes', 'miercoles', 'jueves', 'viernes', 'sabado', 'domingo'];

Para iterar en el arreglo 'semana':

        for (i = 0; i < semana.length; i++){
            console.log(semana[i]);
        }

i se va iterando en cada posición del arreglo y lo imprime con semana[i]

Para interrumpir el ciclo en un punto específico con la sentencia break:

        
        for (i = 0; i < semana.length; i++){
            if(semana[i] == 'miercoles'){
                break;
            }
            document.write(semana[i]);
        }

Para saltar el ciclo en un punto específico y continuar con el ciclo con la sentencia continue:

        
        for (i = 0; i < semana.length; i++){
            if(semana[i] == 'miercoles'){
                continue;
            }
            document.write(semana[i]);
        }

Ciclo 'for in'

Este es un ciclo especial para recorrer arreglos, lleva como parámetros:

        for(variable in arreglo){
            codigo que se repite
        }

Ciclo 'while'

cumple la misma función del ciclo for

        while(condicion){
            código que se repite;
            incremento;
        }
    
        var i = 1;

        while(i < 11){
            console.log(i);
            i++;
        }

inprime los números del 1 al 10

Ciclo 'do while

primero ejecuta una instrucción luego si pasa a verificar la condición

        var i = 1;

        do{
            console.log(i);
            i++;
        }while(i>11);
